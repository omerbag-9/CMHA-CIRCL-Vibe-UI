## Responsive Breakpoints

### Mobile (< 768px)
- Single column layout
- Stacked cards
- Bottom navigation
- Full-width modals
- Simplified tables (card view)

### Tablet (768px - 1024px)
- Two-column layouts where appropriate
- Sidebar becomes drawer
- Optimized table views

### Desktop (> 1024px)
- Full sidebar navigation
- Multi-column layouts
- Expanded table views
- Hover states enabled

## Loading States

### Skeleton Loaders
- **Cards**: Gray rectangles with shimmer animation
- **Tables**: Gray rows with shimmer
- **Text**: Gray lines matching text height
- **Color**: `#2D2D2D` with opacity animation

### Spinner
- **Color**: Green (`#00C853`)
- **Size**: 24px, 32px, 48px variants
- **Animation**: Smooth rotation

---

## Accessibility Features

1. **Keyboard Navigation**: Full keyboard support
2. **Screen Readers**: ARIA labels on all interactive elements
3. **Focus Indicators**: Clear green outline on focus
4. **Color Contrast**: WCAG AA compliant
5. **Text Scaling**: Supports up to 200% zoom
6. **Alt Text**: All images and icons have descriptive alt text

---

## Performance Considerations

1. **Lazy Loading**: Load content as needed
2. **Pagination**: Limit items per page (20-50)
3. **Debounced Search**: Wait for user to stop typing
4. **Optimistic Updates**: Update UI immediately, sync in background
5. **Caching**: Cache frequently accessed data
6. **Image Optimization**: Compress and lazy load images
